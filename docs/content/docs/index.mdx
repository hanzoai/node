# Welcome to Hanzo Node

Hanzo Node is a comprehensive AI infrastructure platform that provides unified access to over 100 LLM providers, advanced tool execution capabilities, and enterprise-grade job orchestration.

## Key Features

- **🤖 100+ LLM Providers** - Unified interface to OpenAI, Anthropic, Google, Meta, and more
- **🔧 8 Runtime Engines** - Execute code in any language (Native, WASM, Docker, Kubernetes, etc.)
- **🔍 LanceDB Vector Search** - Native multimodal storage with vector search capabilities
- **🔐 TEE Attestation** - 5-tier privacy system from Open to Blackwell TEE-I/O
- **💱 Compute Marketplace** - Decentralized exchange for AI compute resources
- **🧠 HLLM Routing** - Physics-based intelligent routing with regime detection
- **⚡ Optimized Performance** - 16-100x improvements with Prometheus monitoring
